# Nature-Nurture
A website for our new SING group.

## Deploy to GitHub Pages

This repo is configured to deploy automatically from GitHub Actions.

### One-time GitHub setup

1. Go to your repository on GitHub.
2. Open **Settings -> Pages**.
3. Under **Build and deployment**, set **Source** to **GitHub Actions**.

### Deploy

1. Commit and push to `main`.
2. Wait for the **Deploy to GitHub Pages** workflow to finish.
3. Your site will be published at:

`https://speshalist.github.io/Nature-Nurture/`
